# Gestao Motoca

Sistema de controle financeiro para motoboys.

O projeto ajuda o entregador a registrar ganhos, despesas, abastecimentos e manutencoes para entender melhor o lucro real do trabalho com a moto.

## Stack

- Backend: Python, FastAPI, SQLAlchemy, Alembic e PostgreSQL.
- Frontend: Vue 3, Pinia, Vue Router, Tailwind CSS e Vite.
- Infra: Docker e Docker Compose.
- Testes: Pytest.
